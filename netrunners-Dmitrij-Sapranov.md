## Опросы

- На второй странице можно добавить вопросы, если нажать на кнопку "Добавить вопрос"

    - Выбрать "Тип вопроса": "Один из списка", "Несколько из списка", "Ответ в произвольной форме" или "Шкала"
    - Для одного или нескольких ответов можно редактировать ответы, добавляя новые, но не менее двух
    - Для типа "Шкала" можно выбрать диапазон
    - Для ответа в произвольной форме доступно только поле для ввода текста
- Можно добавить стоп-экран на определенном вопросе
